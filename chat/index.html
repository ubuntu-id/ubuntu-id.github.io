Chat

Anda dapat bergabung dengan orang lain atau untuk mengajukan pertanyaan dan jawaban.

채팅 서비스 선택

Slack, Matrix (Element.io), IRC (Libera.chat) 을 통해 채팅에 참여하실 수 있습니다. 아래 정보를 참고하여 본인이 사용하기 편한 방식으로 이용해 주세요.

  • Slack: Slack Workspace 접속 (ubuntu-kr.slack.com) | 회원가입
  • Matrix (Element.io): Matrix Space 접속 (#ubuntu-kr) | Element.io 로그인 | Element.io 회원가입
  • IRC (Libera.chat): irc.libera.chat 의 #ubuntu-ko 채널 접속. (웹에서 Kiwi IRC로 바로 접속 하기)

IRC 이용정보

사용하시는 IRC 클라이언트가 있다면, 아래 정보로 IRC 채널에 접속하여 이용할 수 있습니다. IRC 이용시, IRC 이용 규정을 먼저 확인 해 주시기 바랍니다.

  • 서버: irc.libera.chat (또는 irc.ubuntu.com - irc.libera.chat 을 가리키는 주소)
  • 포트
    • 평문 통신: 6665~6667 또는 8000~8002
    • TLS 암호화 통신: 6697, 7000, 7070
  • 채널: #ubuntu-ko
  • IRC 이용 정보 자세히 보기

채널 목록

아래 채널 목록 표를 참고하여 이용해 주시기 바랍니다. 일부 채널은 서로 다른 메신저 서비와 연동 되거나, 대화 내용이 기록되고 있습니다.

채널 용도 Slack Matrix (Element.io) IRC (Libera.chat) 비고
일반적인 대화 및 질의응답 0000-general #ubuntu-ko:libera.chat #ubuntu-ko IRC 대화기록 확인하기*
번역(L10n)팀 채널 0004-l10n #ubuntu-l10n-ko:matrix.org - -
그룹 스터디용 채널 2XXX-* - - -
프로젝트 팀 채널 3XXX-* - - -

*IRC 의 #ubuntu-ko 채널에서의 대화 내역은 우분투 IRC 팀이 운영하는 ubuntulog 봇에 의해 모두 기록되고 있으며, irclogs.ubuntu.com 에서 IRC 대화 기록을 확인하실 수 있습니다.
