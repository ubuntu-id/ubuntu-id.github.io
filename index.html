Komunitas Ubuntu Indonesia

Komunitas Ubuntu Indonesia adalah Ubuntu LoCo (Local Community) di Indonesia. Ini adalah tempat untuk semua orang yang tertarik menggunakan, mengembangkan, dan berkontribusi pada proyek Ubuntu.

Ubuntu 22.04 LTS

Ubuntu 22.04 LTS Jammy Jellyfish adalah versi terbaru yang dirilis pada April 2022. Anda bisa mendapatkannya sekarang, dan menjalankannya di berbagai lingkungan seperti Desktop, Server, IoT, Cloud, dan masih banyak lagi.

Sebelumnya Selanjutnya

Distribusi Linux Paling Populer

Ubuntu adalah salah satu distro Linux yang paling banyak digunakan di dunia. Dikenal sebagai desktop linux yang mudah digunakan, dan mudah untuk mengkonfigurasi distribusi linux untuk server dan infrastruktur lainnya. Ubuntu dioptimalkan untuk berbagai perangkat dan lingkungan seperti Desktop, Laptop, Server, Cloud, Internet of Things, Raspberry Pi, Appliances, Kiosks dan banyak lagi.

Verified Ubuntu LoCo(Local Community)

Ubuntu Indonesia Community is a Ubuntu LoCo(Local Community) in Indonesia verified by Ubuntu Community Council. We provide localized forums and documentations, Plan and host various seminars and conferences. We also organize group studies, research/development projects on Ubuntu and its related open source projects.
